# Shape Calculator

A Java Object-Oriented Programming project that calculates the area of different geometric shapes using abstraction.

## Features

- Calculate area of circles
- Calculate area of rectangles
- Use abstract classes
- Implement method overriding

## OOP Concepts Used

- Abstraction
- Abstract Classes
- Inheritance
- Polymorphism
- Method Overriding

## Technologies Used

- Java
- Object-Oriented Programming (OOP)

## Project Structure

```
ShapeCalculator/
│
├── Main.java
└── README.md
```

## Sample Output

```
Circle Area = 78.53981633974483
Rectangle Area = 40.0
```

## How to Run

### Compile

```bash
javac Main.java
```

### Execute

```bash
java Main
```

## Learning Outcomes

This project demonstrates abstraction by defining common behavior in an abstract class and implementing it in derived classes.
